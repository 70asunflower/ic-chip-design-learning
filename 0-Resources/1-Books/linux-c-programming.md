# Linux C编程一站式学习

## 核心要点（摘要）

经典中文 C 语言教材，从 C 语言基础到 Linux 系统编程一站式覆盖。作者宋劲杉，原书名《Linux C编程一站式学习》。

## 内容覆盖

- **C 语言基础**：数据类型、控制流、函数、指针、数组、字符串
- **C 进阶**：结构体、宏、链接与编译过程、Makefile
- **Linux 系统编程**：文件 I/O、进程、信号、管道、环境变量
- **底层数据表示**：整数的补码表示、浮点数、位运算

## 个人备注

YSYX 项目的 C 语言功底基石——PA 实验要求对指针、内存布局、链接过程有深入理解，这本书把 C 和 Linux 系统知识串起来了。重点看指针章节和编译链接过程部分。
